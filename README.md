# ESP32 Environmental Monitoring System

## Overview
This project is a smart environmental monitoring system based on ESP32. It measures temperature, humidity and light level using sensors and displays the collected data in a real-time web interface.

Based on the detected environmental conditions, the system can also recommend a mood and control audio playback using a DFPlayer Mini module.

## Technologies Used
- ESP32
- Arduino IDE
- DHT22 temperature and humidity sensor
- BH1750 light sensor
- Web API
- DFPlayer Mini module

## Main Features
- Temperature monitoring
- Humidity monitoring
- Light level monitoring
- Real-time data display
- Web interface integration
- Audio playback control
- Hardware-software interaction

## What I Learned
Through this project, I improved my understanding of embedded systems, sensor integration, microcontroller programming and communication between hardware and web-based applications.
