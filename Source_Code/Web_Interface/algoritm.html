Algoritm AI

Algoritmul AI are rolul de a interpreta valorile citite de la senzori și de a determina starea ambientală a încăperii. Pe baza acestei stări, sistemul recomandă automat o melodie potrivită pentru utilizator.

🧠 Date analizate

🌡️ Temperatură

💧 Umiditate

💡 Nivel de lumină

⚙️ Clasificare ambientală

Valorile sunt comparate cu un set de scenarii de antrenare, iar sistemul determină cea mai apropiată stare ambientală.

🎵 Recomandare muzicală

În funcție de starea detectată, sistemul selectează automat o melodie potrivită: relaxare, concentrare, energie sau alertă.

📌 Exemplu

Dacă temperatura este confortabilă, umiditatea este normală și lumina este redusă, sistemul poate detecta starea de relaxare și poate recomanda o melodie ambientală.
