Descriere proiect

Proiectul propus este un sistem inteligent de monitorizare a condițiilor ambientale, realizat cu ajutorul microcontrolerului ESP32. Sistemul citește valori precum temperatura, umiditatea și nivelul de lumină dintr-o încăpere, apoi afișează aceste date pe o interfață web.

Pe baza valorilor citite de senzori, aplicația analizează starea ambientală și recomandă automat o melodie potrivită. Astfel, proiectul combină partea hardware cu partea software, oferind o experiență interactivă și ușor de urmărit.

Scopul proiectului este de a demonstra modul în care datele preluate de la senzori pot fi folosite pentru luarea unor decizii automate, într-un sistem simplu de tip smart environment.
