# Source Code

This folder contains the complete source code for the ESP32 Environmental Monitoring System.

The project is divided into two main components:

- **Arduino_Code** – ESP32 firmware developed using Arduino IDE, responsible for reading sensor data, controlling the hardware and communicating with external services.
- **Web_Interface** – HTML, CSS and JavaScript files that provide a real-time monitoring dashboard and user interface for the system.

Together, these components form a complete IoT solution that integrates hardware, software and web technologies.
